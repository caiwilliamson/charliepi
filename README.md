# Charlie Pi 🐈‍⬛🥧

### Install
With [pipenv][pipenv]:
```
pipenv install --site-packages
```

### Install daemons
This will install the systemd services under `daemons` and start them
```
pipenv run install_daemons
```

### Run web app
```
pipenv run web
```

[pipenv]: https://github.com/pypa/pipenv
